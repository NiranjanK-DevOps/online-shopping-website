# Online Shopping Website (Firebase + S3)

A basic online shopping UI that fetches product data from Firebase Firestore and displays it on a static website hosted using AWS S3.

## Features
- Fetches product details dynamically from Firestore
- Displays product image, name, and price
- Clean and simple UI
- Fully static frontend hosted on S3

## Technologies Used
- HTML
- CSS
- JavaScript
- Firebase Firestore
- AWS S3

## Purpose
To learn:
- Using Firestore database in a frontend app
- Connecting static websites to cloud databases
- Hosting production-ready static sites on S3
